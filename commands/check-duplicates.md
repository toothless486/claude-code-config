# 중복 코드 검사 Command

지정된 파일 또는 디렉토리에서 중복 코드를 분석한다.

## 분석 절차

### 1. 대상 파일 수집
```bash
find $ARGUMENTS -type f \( -name "*.c" -o -name "*.cpp" -o -name "*.h" -o -name "*.hpp" \)
```

### 2. 중복 패턴 유형

#### 타입 A: 복사-붙여넣기 중복
- 거의 동일한 코드 블록이 여러 파일에 존재
- **해결**: 공통 함수로 추출

#### 타입 B: 구조적 중복
- 같은 패턴이 타입/변수만 다르게 반복
- **해결**: 매크로 또는 템플릿 (C++)

#### 타입 C: 로직 중복
- 다른 코드지만 같은 목적 수행
- **해결**: 인터페이스 통일

### 3. 분석 관점

각 함수에 대해:
1. 유사한 함수명 검색 (`grep -r "함수패턴"`)
2. 유사한 로직 패턴 검색
3. 공통 에러 처리 / NULL 체크 패턴 확인

### 4. 출력 형식

```markdown
## 중복 코드 분석 결과

### 발견된 중복 (총 N건)

#### 1. [중복 유형] 설명
- 위치: file1.c:10-25, file2.c:30-45
- 유사도: 높음/중간
- 제안: 구체적인 리팩토링 방법

### 공통화 우선순위
1. (높음) ...
2. (중간) ...
3. (낮음) ...
```

## 입력
$ARGUMENTS
